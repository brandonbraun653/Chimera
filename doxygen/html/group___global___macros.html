A 'factory-like' macro that generates a compile time check for the existence of a specific function signature within a given class. Code is not generated in flash and there is no overhead. Mostly this function is used in the various Chimera core perihperal module namespaces like GPIO, UART, SPI, etc. to give the developer an error. If desired, a runtime check could be used simply by calling "InvocationName::value".
